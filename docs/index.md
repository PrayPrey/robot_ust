## Getting Started

### Quick Start

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Configure environment
cp .env.template .env
# Edit .env with OPENAI_API_KEY

# 3. Install Ollama
ollama pull tinyllama

# 4. Start web server
uvicorn src.web.server:app --reload

# 5. Open browser
# http://localhost:8000
```
